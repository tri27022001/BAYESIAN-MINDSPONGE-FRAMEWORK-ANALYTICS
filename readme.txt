Hướng dẫn chạy dự án

1. Yêu cầu hệ thống

Python (>=3.7)

Visual Studio Code hoặc Google Colab

Các thư viện Python cần thiết

2. Cài đặt các thư viện cần thiết

Chạy lệnh sau trong terminal hoặc command prompt để cài đặt các thư viện cần thiết:

pip install pandas numpy jupyter, plt
Trong file code có một cell có câu lệch dung để cài thư viện pgmpy rồ nên chỉ chỉ cần chạy cell đó là được.
3. Chạy dự án

3.1 Sử dụng Visual Studio Code

Mở Visual Studio Code.

Cài đặt tiện ích mở rộng "Python" và "Jupyter" nếu chưa có.

Mở tệp cntt2.ipynb trong VS Code.

Chạy từng ô mã (cell) bằng cách nhấn Shift + Enter.

3.2 Sử dụng Google Colab

Truy cập Google Colab.

Tải tệp cntt2.ipynb lên Google Colab.

Chạy từng ô mã (cell) bằng cách nhấn Shift + Enter

4. Dữ liệu đầu vào

Dự án dùng tệp dữ liệu allData.csv.

Link tải data: https://drive.google.com/file/d/1d5cLVGQQtN-SoDpgw8yeLATmrCiVNDBE/view?usp=sharing

Đảm bảo tệp này nằm cùng thư mục với tệp notebook cntt2.ipynb (nếu dùng VS Code).


Nếu sử dụng Google Colab, tải tệp allData.csv lên hoặc sử dụng Google Drive để lưu trữ dữ liệu.
